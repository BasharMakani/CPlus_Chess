# C++ Terminal Chess Game

## Overview
A fully functional terminal-based chess game developed in C++. The project implements core chess mechanics, including piece movement, rule enforcement, and turn-based gameplay, using object-oriented programming principles.

## Features
- Complete chess board representation  
- Legal move validation for all pieces  
- Turn-based gameplay system  
- Capture mechanics and game state tracking  
- Input-driven command system for player moves  
- Structured using object-oriented design  

## Technologies
- C++  
- Object-Oriented Programming (OOP)  

## How to Run
1. Clone the repository:
   git clone https://github.com/yourusername/cpp-chess.git  

2. Navigate to the project directory  

3. Compile the program:
   g++ main.cpp -o chess  

4. Run the executable:
   ./chess  

## Gameplay
- Players take turns entering moves via terminal input  
- Moves follow standard chess rules  
- The game updates the board after each move
